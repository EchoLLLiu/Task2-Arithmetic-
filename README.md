# Task2-Arithmetic
### 1.项目功能说明：<br>
- 随机生成100以内的整数
- 随机生成真分数
- 随机生成1~10个运算符（扩展功能）
- 使用 -n 参数控制生成题目的个数
- 能够处理用户的输入，并对每道题的用户输入进行对错的判断
- 进行正确率的统计以及分数评定<br>
### 2.项目操作说明：<br>
- 该项目有两个文件：arithmetic.py函数文件以及start.py启动文件
- 测试文件：arith_test.py
- 使用控制台运行该文件：<br>
输入python start.py -h，查询该程序帮助文档<br>
输入python start.py -n x，x为想要生成的题目个数。
